#jQuery Tooltip plugin 

## How to Use?

jQuery Tooltip plugin depends on jQuery. Include script *after* the jQuery library:

```html
<script src="jquery.js" type="text/javascript"></script>
<script src="tooltip.js" type="text/javascript"></script>
```
